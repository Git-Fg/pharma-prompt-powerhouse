Le **prompting avec XML** est une méthode de structuration qui utilise des balises (comme en HTML) pour délimiter clairement les différentes parties d'un prompt (contexte, instructions, format de sortie). C'est la méthode recommandée par Anthropic pour les modèles Claude, car elle améliore considérablement la fiabilité des réponses.

## Pourquoi utiliser le XML Prompting ?

Cette approche structurée offre plusieurs avantages :

- **Clarté maximale** : L'IA comprend exactement ce que vous attendez
- **Réduction des hallucinations** : Les balises guident le modèle vers des réponses précises
- **Formatage cohérent** : Les sorties respectent toujours la structure demandée
- **Facilité de maintenance** : Les prompts sont plus lisibles et modifiables

## Structure de base

Un prompt XML typique se compose de :

- `<context>` : Informations de base et contexte
- `<instructions>` : Ce que vous voulez que l'IA fasse
- `<format>` : Comment vous voulez la réponse
- `<examples>` : Exemples pour clarifier vos attentes

## Applications en pharmacie

- **Analyse de cas cliniques** : Structurer les informations patient et les questions
- **Génération de rapports** : Assurer un formatage cohérent des documents
- **Évaluation de risques** : Organiser les facteurs de risque et les recommandations
- **Planification thérapeutique** : Structurer les étapes du traitement

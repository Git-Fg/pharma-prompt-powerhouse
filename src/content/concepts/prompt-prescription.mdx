# Prompt : Prescription

## L'analogie parfaite

Un **prompt** est à l'IA ce qu'une **prescription** est au patient. Cette analogie vous aidera à structurer systématiquement vos demandes pour obtenir des résultats prévisibles et de qualité.

## Les 5 éléments d'une prescription parfaite

### 1. Le Rôle (La Molécule)

**Qu'est-ce que c'est ?** L'identité et la spécialité de l'IA
**Exemple :**

```
Tu es un pharmacien clinicien spécialisé en gériatrie,
habitué à optimiser des ordonnances complexes pour des
patients polymédiqués en EHPAD.
```

### 2. La Tâche (La Posologie)

**Qu'est-ce que c'est ?** L'action précise à effectuer
**Exemple :**

```
Crée une fiche de révision synthétique sur la classe des IEC.
La fiche doit être au format Markdown et optimisée pour
un étudiant de 5ème année en pharmacie.
```

### 3. Le Contexte (Les Antécédents)

**Qu'est-ce que c'est ?** Les informations pertinentes pour la prise de décision
**Exemple :**

```
Voici une ordonnance pour un patient de 75 ans avec une
insuffisance rénale (ClCr 40 mL/min). Analyse-la en te
basant sur le référentiel de la HAS de 2023 fourni ci-dessous.
```

### 4. Le Format (La Voie d'Administration)

**Qu'est-ce que c'est ?** La forme de la réponse attendue
**Exemple :**

```
Présente les effets indésirables dans un tableau Markdown
à 3 colonnes : "Effet", "Fréquence (RCP)", et "Gestion/Conseil".
```

### 5. Les Exemples (Les Cas Cliniques de Référence)

**Qu'est-ce que c'est ?** Des modèles concrets de ce qui est attendu
**Exemple :**

```
Transforme les informations suivantes en une carte Anki.
Voici un exemple du format attendu :
Recto: "Quel est le principal risque de l'association IEC + Spironolactone ?"
Verso: "Hyperkaliémie (synergie des effets hyperkaliémiants)."
```

## Pourquoi cette analogie fonctionne ?

### Précision

Comme une prescription, un prompt bien rédigé ne laisse aucune place à l'ambiguïté. Chaque élément a sa place et sa fonction.

### Complétude

Une prescription oubliée peut avoir des conséquences graves. De même, un prompt incomplet donnera des résultats imprévisibles.

### Reproductibilité

Un bon prompt, comme une bonne prescription, peut être réutilisé et adapté à d'autres situations similaires.

## Applications pratiques

### Cas 1 : Analyse d'interaction médicamenteuse

```
[RÔLE] Tu es un pharmacien clinicien expert en interactions médicamenteuses
[TÂCHE] Analyse cette association et propose un plan de monitoring
[CONTEXTE] Patient de 70 ans, clairance créatinine 35 ml/min
[FORMAT] Tableau avec risque, monitoring, alternatives
```

### Cas 2 : Création de fiche de révision

```
[RÔLE] Tu es un enseignant en pharmacologie
[TÂCHE] Crée une fiche de révision sur les antiagrégants plaquettaires
[CONTEXTE] Niveau L3, format concis
[FORMAT] Markdown avec sections numérotées
```

## Vérification de qualité

Avant d'envoyer votre prompt, vérifiez qu'il contient bien :

- ✅ Un rôle clair et spécifique
- ✅ Une tâche précise et actionnable
- ✅ Un contexte suffisant mais pas excessif
- ✅ Un format de sortie défini
- ✅ Des exemples si nécessaire

## Bénéfices de cette approche

1. **Résultats plus prévisibles** : L'IA sait exactement ce qu'elle doit faire
2. **Moins d'itérations** : Le prompt est complet dès le départ
3. **Apprentissage progressif** : Vous améliorez votre technique prompt par prompt
4. **Réutilisabilité** : Un bon prompt peut être adapté à d'autres cas

En adoptant cette approche "prescription", vous transformez l'art du prompting en une science précise et reproductible, exactement comme la pharmacie clinique.

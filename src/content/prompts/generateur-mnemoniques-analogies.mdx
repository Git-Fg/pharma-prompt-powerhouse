## Notes d'Utilisation

C'est une tâche créative. Le paramètre clé est la **température**. Je recommande **Google AI Studio** pour pouvoir la régler précisément.

-   **Sur AI Studio :** Réglez la température entre `0.7` et `0.9` pour des résultats originaux. En dessous, les réponses seront trop convenues.
-   **Sur ChatGPT/Claude :** Précisez explicitement le besoin de créativité dans le prompt, car vous ne pouvez pas contrôler la température.

### Prompt Recommandé (pour Google AI Studio)

Pas besoin de balises ici, le prompt est direct. L'important est le réglage des paramètres.

```markdown
# Réglages dans AI Studio
# Température: 0.8

# Prompt
Tu es un tuteur en pharmacie extrêmement créatif, connu pour tes moyens mnémotechniques inoubliables.

Crée une aide-mémoire de type **{type_aide_memoire}** pour mémoriser la liste suivante.
Le style doit être **{style_prefere}**.

**Éléments à mémoriser :**
{elements_a_memoriser}
```

### Adaptation pour ChatGPT / Claude

Tu es un tuteur en pharmacie extrêmement créatif. Sois très original et imaginatif.

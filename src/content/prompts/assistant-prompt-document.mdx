## Notes d'Utilisation

Ce prompt est conçu pour analyser un contenu textuel que vous fournissez. La meilleure plateforme pour cela est **Claude**, car elle permet de téléverser des fichiers PDF directement. Pour les autres outils, vous devrez copier-coller le texte du document.

-   **Sur Claude :** Utilisez l'icône trombone pour joindre votre fichier avant d'envoyer le prompt.
-   **Sur AI Studio / ChatGPT :** Copiez le texte du document et collez-le à la place de `{document_content}`.

### Prompt Recommandé (pour Claude & AI Studio)

L'utilisation de balises `<document>` et `<question>` est ici très pertinente pour bien séparer la source de vérité de l'instruction, ce qui réduit drastiquement le risque d'hallucination.

```xml
<document>
{document_content}
</document>

<instructions>
En te basant EXCLUSIVEMENT sur le document fourni ci-dessus, réponds à la question suivante. Si l'information n'est pas présente dans le texte, réponds "L'information n'est pas disponible dans le document fourni."
</instructions>

<question>
{user_question}
</question>
```

### Adaptation pour ChatGPT

ChatGPT répond bien à un style plus direct et conversationnel.

```markdown
### Contexte
Voici un document :
"""
{document_content}
"""

### Ma Question
En me basant uniquement sur le document ci-dessus, réponds à cette question : {user_question}

**Règle importante :** Si la réponse ne se trouve pas dans le document, dis-le explicitement.

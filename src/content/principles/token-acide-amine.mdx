---
title: "Token = Acide Aminé"
description: "Comprendre le token comme la brique de base des modèles de langage."
---

Un token est la brique de base de l'IA, comme un acide aminé pour une protéine. Les tokens s'assemblent pour former des phrases.

## Qu'est-ce qu'un Token ?

Un token peut être :
- Un mot entier (ex: "pharmacien")
- Une partie de mot (ex: "anti" et "coagulant")
- Un caractère de ponctuation (ex: ".", ",")

## Impact sur les Prompts

Le nombre de tokens dans un prompt et sa réponse influence :
- La complexité des réponses possibles
- Le coût de la requête (pour les API payantes)
- Les limites de contexte (la "mémoire" de l'IA)

## Optimisation

Pour optimiser l'utilisation des tokens :
- Soyez concis et précis
- Évitez les répétitions inutiles
- Utilisez des formats structurés comme Markdown pour maximiser la densité d'information

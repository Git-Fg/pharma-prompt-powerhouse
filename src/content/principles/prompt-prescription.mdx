---
title: "Prompt = Prescription"
description: "Analogie entre la rédaction d'un prompt et la prescription d'un médicament."
---

Votre prompt est une ordonnance pour l'IA. Cette analogie nous aide à comprendre comment structurer efficacement nos instructions.

## Les Composantes de la Prescription

### Molécule → Le Rôle
Qui doit être l'IA ? Définissez clairement la spécialité et l'expertise attendue.
- *Exemple :* "Tu es un professeur de pharmacologie spécialisé en cardiologie."

### Posologie → La Tâche
Que doit-elle faire ? Utilisez des verbes d'action précis pour décrire l'objectif.
- *Exemple :* "Crée une fiche de révision synthétique sur les bêta-bloquants."

### Voie d'administration → Le Format
Comment doit-elle présenter la réponse ? Spécifiez le format de sortie attendu.
- *Exemple :* "Présente les informations sous forme de tableau Markdown avec les colonnes : Nom, Mécanisme d'action, Effets indésirables."

## Application Pratique

Comme une prescription bien rédigée, un prompt efficace contient toutes les informations nécessaires pour que l'IA produise exactement le résultat souhaité, sans ambiguïté.

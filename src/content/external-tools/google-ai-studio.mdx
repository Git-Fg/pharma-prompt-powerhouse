Google AI Studio est l'interface web officielle de Google pour interagir avec les modèles Gemini 2.5. C'est l'outil parfait pour les étudiants en pharmacie qui veulent maîtriser l'ingénierie de prompts sans configuration complexe.

## Points forts

- **Gratuit et accessible** : Aucun compte Google Cloud requis
- **Modèles Gemini 2.5** : Accès aux dernières avancées en IA
- **Interface intuitive** : Conçue pour les débutants
- **Paramètres avancés** : Contrôle précis du comportement des modèles
- **Outils spécialisés** : Structured Output, Code Execution, Grounding

## Utilisation recommandée

Google AI Studio est idéal pour :

- L'apprentissage de l'ingénierie de prompts
- L'expérimentation avec différents paramètres
- La création de prompts structurés pour la pharmacie
- L'analyse de cas cliniques complexes

## Alternative à

- Vertex AI Studio (plus complexe, orienté entreprise)
- ChatGPT (moins de contrôle sur les paramètres)
- Claude (interface différente, moins d'intégration Google)

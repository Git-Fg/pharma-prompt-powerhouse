## 🔬 Google AI Studio : Le Laboratoire Gratuit de Google

Google AI Studio est l'interface **développeur avancée** de Google pour les modèles Gemini. Contrairement au chat Gemini standard, c'est votre paillasse de laboratoire pour expérimenter avec tous les paramètres et fonctionnalités avancées.

## 🆚 Différences avec Gemini Standard

### Gemini (gemini.google.com)
- ✅ **Gratuit** : Gemini 2.5 Flash, limité à 32K tokens, 10 fichiers max 100Mo
- ✅ Interface conversationnelle intuitive
- ✅ Parfait pour l'usage quotidien
- ✅ Upload d'images et fichiers PDF
- ❌ Paramètres fixes (pas d'accès aux réglages)
- ❌ Un seul modèle accessible (Gemini 2.5 Flash)
- ❌ Pas de system prompt personnalisé
- **Payant (20€/mois)** : Gemini 2.5 Pro, 1M tokens, 100 requêtes/jour, Deep Research, Veo 3

### Google AI Studio (aistudio.google.com)
- ✅ **API gratuite avec quotas généreux** : 60 requêtes/min, 300K tokens/jour
- ✅ **Tous les modèles Gemini** : 2.5 Pro, 2.5 Flash, 2.5 Flash-Lite, Imagen 4
- ✅ **Contrôle total des paramètres** (température, top-k, top-p)
- ✅ **System Prompt** personnalisé et persistant
- ✅ **Thinking Mode** pour voir le raisonnement (Gemini 2.5 Pro)
- ✅ **Fonctionnalités avancées** : structured outputs, function calling, code execution
- ✅ **Support multimodal** : texte, image, audio, vidéo, PDF
- ✅ **URL context et search grounding**
- ✅ **Caching support** pour optimiser les performances
- ✅ **Aucune CB nécessaire** - idéal pour étudiants
- ❌ Interface plus complexe

## 🎯 Pourquoi Choisir AI Studio ?

### Avantages Uniques pour la Pharmacie
- **Control total du "dosage"** : ajustez la précision vs créativité
- **System prompts persistants** : définissez une expertise une fois pour toutes
- **Thinking Mode** : validez le raisonnement de l'IA étape par étape
- **Accès aux derniers modèles** dès leur sortie
- **Totalement gratuit** contrairement aux autres playgrounds

## ⚙️ Paramètres Clés à Maîtriser

### Température (0.0 - 2.0)
```
❄️  Temperature = 0.0-0.2
→ Maximum de précision et reproductibilité
→ Idéal pour : calculs, diagnostics, analyses factuelles

🌡️ Temperature = 0.3-0.7  
→ Équilibre créativité/précision
→ Idéal pour : explications, synthèses pédagogiques

🔥 Temperature = 0.8-2.0
→ Maximum de créativité
→ Idéal pour : analogies, mnémotechniques, brainstorming
```

### System Prompt : Votre Expertise Permanente
```xml
<system>
Tu es un pharmacien hospitalier expert en interactions médicamenteuses.

MÉTHODE D'ANALYSE :
1. Identification des mécanismes pharmacologiques
2. Évaluation du risque clinique (faible/modéré/élevé)
3. Recommandations pratiques avec alternatives

STYLE DE RÉPONSE :
- Pédagogique et structuré
- Toujours justifié scientifiquement
- Adapté aux étudiants en pharmacie
</system>
```

### Thinking Mode : Le Raisonnement Visible
Activé, il force Gemini à "penser à voix haute" - essentiel pour valider ses analyses.

### 2. Workflow pour une Analyse de Cas Clinique

**Étape 1 : System Prompt**
```
Tu es un expert en pharmacovigilance. 

Ton analyse est systématique, rigoureuse et toujours justifiée par des mécanismes pharmacologiques. 

Tu hiérarchises les risques du plus élevé au plus faible et proposes des alternatives pratiques.

STRUCTURE OBLIGATOIRE :
1. Analyse du risque
2. Mécanisme impliqué  
3. Recommandations pratiques
```

**Étape 2 : Prompt Utilisateur Structuré**
```xml
<cas>
  <patient>Femme, 68 ans, insuffisance rénale modérée (DFG: 35 mL/min)</patient>
  <traitement>Atorvastatine 40mg/j, Amlodipine 10mg/j</traitement>
  <nouveau_medicament>Clarithromycine 500mg x2/j pour pneumonie</nouveau_medicament>
</cas>

<question>
Analyse l'interaction potentielle. Détaille le mécanisme, évalue le risque clinique et propose un plan d'action.
</question>
```

**Étape 3 : Validation du Raisonnement**
Avec Thinking Mode activé, vous verrez le processus de réflexion de Gemini, permettant de valider chaque étape.

## 🔥 Fonctionnalités Avancées

### Structured Output
Forcez Gemini à répondre dans un format JSON ou XML précis :
```json
{
  "interaction": {
    "medicaments": ["atorvastatine", "clarithromycine"],
    "mecanisme": "Inhibition CYP3A4",
    "risque": "élevé",
    "symptomes": ["rhabdomyolyse", "myalgies"],
    "recommandation": "Surveillance CPK + arrêt temporaire statine"
  }
}
```

## 💡 Astuces pour Optimiser vos Études

### 1. Créez des System Prompts Spécialisés
- Un pour la **pharmacologie** (mécanismes détaillés)
- Un pour la **galénique** (formulations et stabilité)  
- Un pour les **cas cliniques** (approche diagnostique)

### 2. Utilisez les Balises XML
Structure vos prompts pour plus de clarté :
```xml
<contexte>Cours de pharmacocinétique</contexte>
<objectif>Comprendre la clairance rénale</objectif>
<niveau>Étudiant 4ème année</niveau>
<question>Explique l'impact de l'âge sur la clairance de la créatinine</question>
```

- Comprendre la logique derrière une recommandation
- Apprendre la démarche de résolution de problème
